# 📸 Instagram Caption Generator Pro

An AI-powered Instagram caption generator built with Python, Streamlit, and Groq API.

## ✨ Features

- 🎯 16+ Topics (Cars, Travel, Fashion, Fitness, etc.)
- 🎨 12+ Styles (Funny, Luxury, Motivational, etc.)
- ⭐ Save Favorites
- 📜 Generation History
- 📥 Download Captions (TXT/CSV)
- 📝 Character Counter
- 🌐 Multiple Languages
- 🎚️ Creativity Control

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/instagram-caption-generator.git

# Install dependencies
pip install -r requirements.txt

# Run the app
cd app
streamlit run streamlit_app.py
