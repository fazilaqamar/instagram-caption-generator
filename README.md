# 📸 AI-Powered Instagram Caption Generator

An intelligent Instagram caption generator that uses Groq's Llama 3.1 AI model to create engaging, viral-worthy captions.

## ✨ Features

- 🎯 **16+ Topics**: Cars, Travel, Fashion, Fitness, Food, Nature, Tech, Beauty, Pets, Music, Art, Photography, Business, Entrepreneur, Lifestyle, Wellness
- 🎨 **12+ Styles**: Funny, Luxury, Motivational, Aesthetic, Bold, Emotional, Savage, Relatable, Poetic, Witty, Inspirational, Vibes
- ⭐ **Save Favorites**: Save your favorite captions
- 📜 **Generation History**: View your recent generations
- 📥 **Download Options**: Export as TXT or CSV
- 📝 **Character Counter**: Track caption length
- 🌐 **Multi-language**: English, Spanish, French, German, Hindi, Urdu, Arabic
- 🎚️ **Creativity Control**: Adjust AI creativity level

## 🚀 Live Demo

[View Live Demo](https://fazila-caption-generator.streamlit.app)

## 🛠️ Tech Stack

- **Python 3.11** - Core programming language
- **Streamlit** - Web application framework
- **Groq API** - Llama 3.1 AI model
- **Pandas** - Data analysis
- **Matplotlib** - Data visualization
